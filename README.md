# Google Books Search

Demo: https://frozen-woodland-04984.herokuapp.com/

This is a simple MERN stack application which uses the Google Books API to search for books. 

Users can save books to a list to view later.

- React.js
- MongoDB
- Express.js
- Node.js
- React-router
- CSS3
- Axios
- Bootstrap
